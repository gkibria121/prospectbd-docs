## Fields

| Field Name   | Type    | Description                          |
|--------------|---------|--------------------------------------|
| Worker Name  | String  | The name of the worker               |
| Task ID      | Integer | ID of the assigned task              |

## Actions

- **Update Task**: Updates the status or completion of the task.
