## Fields

| Field Name       | Type    | Description                 |
|------------------|---------|-----------------------------|
| ServiceID        | Integer | ID of the inquired service   |
| Inquiry Message  | Text    | Message or question related to the service |

## Actions

- **Submit Inquiry**: Submits the inquiry form to the service provider.
