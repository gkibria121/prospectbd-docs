## Fields

| Field Name     | Type    | Description                        |
|----------------|---------|------------------------------------|
| ServiceID      | Integer | ID of the service requested        |
| Estimated Cost | Decimal | Estimated cost for the requested service |

## Actions

- **Request Quote**: Submits the quote request.
